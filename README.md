# 🚴 Bike Sharing Dashboard ✨

> Analisis data penyewaan sepeda Capital Bikeshare, Washington D.C. (2011–2012)

---

## 📋 Deskripsi

Dashboard interaktif untuk menjawab pertanyaan bisnis:
1. Bagaimana pengaruh kondisi cuaca dan musim terhadap jumlah penyewaan sepeda harian?
2. Pada jam berapa dan hari apa penyewaan sepeda mencapai puncaknya dalam seminggu?

---

```
submission

```

---

## ⚙️ Setup Environment - Shell/Terminal

```bash
mkdir proyek_analisis_data
cd proyek_analisis_data
pip install -r requirements.txt
```

---

## ▶️ Run Streamlit App

```bash
streamlit run dashboard/dashboard.py
```
